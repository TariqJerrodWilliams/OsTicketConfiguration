<h1 align="center">osTicket - Post-Install Configuration</h1>
<p align="center">This is a demonstration showing how to configure Users, Teams, Departments, SLAs, and Help Topics in OsTicket.</p>
<br />


<h2 align="center">Video Demonstration</h2>

<p align="center"><a href="https://youtu.be/34fYHe6vFNE">Configuration Of OsTicket</a></p>
<h2 align="center">Environments and Technologies Used</h2>

<p align="center">Microsoft Azure (Virtual Machines/Compute)</li>
<p align="center">Remote Desktop</li>
<p align="center">Internet Information Services (IIS)</li>

<h2 align="center">Operating Systems Used </h2>

<p align="center">Windows 10</b> (21H2)</p>

<h2 align="center">Post-Install Configuration Objectives</h2>

<p align="center">- We need to build a team of professionals to manage tickets.</p>
<p align="center">- We need to establish protocols, adhering to priority, to sort tickets efficiently.</p>
<p align="center">- We need to prepare categories for users to choose from so that we can organize and direct them to the correct department.</p>
